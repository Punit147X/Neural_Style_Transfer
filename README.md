# Neural Style Transfer

## Overview
This project applies the artistic style of one image to another using Neural Style Transfer with TensorFlow Hub.

## Features
- Loads a content image and a style image
- Uses a pre-trained TensorFlow Hub model
- Generates a stylized output image
- Saves the final output automatically

## Technologies Used
- Python
- TensorFlow
- TensorFlow Hub
- NumPy
- Pillow
- Matplotlib

## Files
- Neural_Style_Transfer.ipynb
- content.jpg
- style.jpg
- output.png

## How to Run
1. Upload `content.jpg` and `style.jpg` to Google Colab.
2. Run all notebook cells.
3. The stylized image will be saved as `output.png`.

## Result
The model successfully transfers the artistic style from the style image to the content image using deep learning.
